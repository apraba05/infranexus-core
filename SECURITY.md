# Security Policy

## Supported Versions

We only provide security updates for the current major release of InfraNexus.

| Version | Supported          |
| ------- | ------------------ |
| v1.x    | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

Security is a core focus for InfraNexus. We take all security vulnerabilities seriously.

To report a vulnerability, please email **security@infranexus.com** rather than creating a public GitHub issue. We strive to acknowledge reports within 48 hours and will keep you informed of our progress towards a fix and an announcement.

Please include:
- A detailed description of the vulnerability.
- Steps to reproduce.
- Any potential impact.

**Do not publicly disclose the vulnerability until we have had a chance to patch it and notify users.**
